# Dashboard UI — Frontend Assignment

A responsive Dashboard UI built with **Next.js** and **Ant Design**, based on the provided Figma design.

---

## 🛠 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **UI Library:** Ant Design
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Icons:** Lucide React
- **Charts:** Recharts
- **Animations:** Framer Motion

---

## 📦 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/Rasujon3/easy-job-task.git
cd easy-job-task
```

2. **Install dependencies**

```bash
npm install
```

3. **Run the development server**

```bash
npm run dev
```

4. **Open in browser**

```
http://localhost:3000
```

---

## 🏗 Build for Production

```bash
npm run build
npm run start
```
